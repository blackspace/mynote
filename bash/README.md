为什么变量名要双引号引用
--------------------------------------

```
#!/usr/bin/env bash

echo $(pwd)

a="b*"

#用双引号阻止路径名扩展
echo "$a"


#如果不用双引号,会列出当前目录下符合模式的文件名
echo $a
```


bash的3种引用机制
------------------------------------------
* escape character
* single quotes
* double quotes

$  '  \有特定的意义，在双引号中保留$和'的特定意义,以及当\后跟$ ' " \ <newline>时的特定意义。
